# Model Artifacts

Place the demo model files you want to use here when preparing the Jetson run.

Typical files:
- `baseline_int8_int8.engine`
- `best_s2n_v2_int8.engine`
- `best_m2n_v2_int8.engine`
- `best_l2n_v2_int8.engine`
- `yolov8n.pt`

This folder is intended as a convenient local drop location for demo-time artifacts.
